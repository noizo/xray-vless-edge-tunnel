# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately using
[GitHub's private vulnerability reporting](https://github.com/noizo/xray-vless-edge-tunnel/security/advisories/new).

Do **not** open a public issue for security vulnerabilities.

## Scope

This policy covers:

- The `xray-admin` Go application (`cmd/admin/`)
- Kubernetes deployment manifests (`deployments/kubernetes/`)
- CI/CD workflows (`.github/workflows/`)
- Shell scripts (`scripts/`)

## Response

- Acknowledgement within 48 hours
- Fix or mitigation within 7 days for critical issues
